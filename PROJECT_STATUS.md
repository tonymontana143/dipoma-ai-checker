## 🚀 НАЧНИТЕ ОТСЮДА

### Шаг 1: Активируйте окружение
```bash
source venv/bin/activate
```

### Шаг 2: Обучите модель (первый раз)
```bash
python train.py
```
⏱️ **Время**: ~30 минут на GPU, ~1-2 часа на CPU

### Шаг 3: Протестируйте результаты
```bash
# Демонстрационный режим с примерами
python test_model.py

# Или анализ конкретного текста
python test_model.py "Ты дебил!"
```
