## 🎯 Шаг за шагом

### 1️⃣ Активируйте виртуальное окружение

```bash
source venv/bin/activate
```

### 2️⃣ Запустите обучение модели

```bash
python train.py
```

**Что произойдет**:
- Загрузится `labeled.csv` (русский датасет с комментариями)
- Данные разделятся на обучающее и валидационное множества
- Модель RuBERT обучится на 3 эпохи
- Модель будет сохранена в папку `toxic_comment_model/`

**Ожидаемое время**:
- На GPU: 15-30 минут
- На CPU: 1-2 часа

### 3️⃣ Протестируйте обученную модель

```bash
# Демонстрационный режим с примерами
python test_model.py

# Или анализ конкретного текста
python test_model.py "Ты дебил!"
```

## 📊 Структура данных

Файл `labeled.csv` содержит:
- **comment** - текст комментария на русском языке
- **toxic** - метка (1.0 = токсичный, 0.0 = безопасный)

Пример:
```csv
comment,toxic
"Ты идиот!",1.0
"Спасибо за статью!",0.0
```

## 🤖 Используемая модель

**DeepPavlov/rubert-base-cased**
- Специально обучена на русском языке
- 109M параметров
- Проверенная на токсичности комментариев

## 📈 Ожидаемые метрики

После обучения на полном датасете:
- **Accuracy** > 0.92
- **F1 Score** > 0.85
- **ROC-AUC** > 0.95

## 🆘 Возможные проблемы

### ❌ Out of Memory (OOM)

**Решение**: Уменьшить batch_size в `train.py`:
```python
CONFIG['batch_size'] = 4  # вместо 8
```

### ❌ Обучение медленное

**Решение**: 
- Уменьшить размер датасета в `train.py`
- Использовать Google Colab (бесплатно с GPU)

### ❌ Модель не найдена при тестировании

**Решение**: Сначала обучите модель (шаг 2)
